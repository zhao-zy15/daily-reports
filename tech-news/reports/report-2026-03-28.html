Anthropic 意外泄露 Claude Mythos 模型，确认正在测试"性能阶跃式提升"新模型 LLM

📅 2026-03-27 · 📰 Fortune

事件背景：Anthropic 一篇未发布的博客文章被发现存储在一个未受保护的数据存储库中，文中提到了名为 Claude Mythos 的新模型。泄露文档显示，该模型代表着 Anthropic 能力的"阶跃式变化（step change）"，而非渐进式改进。

🔮 产业影响：这一泄露事件既暴露了 Anthropic 在数据安全方面的短板，也预示着 Claude 产品线即将迎来重大更新。若 Mythos 兑现"阶跃提升"承诺，将直接冲击 GPT-5.4 和 Gemini 3.1 Pro 的竞争格局。

Anthropic 发布第二份经济指数报告：AI 技能"学习曲线"效应显著 研究

📅 2026-03-28 · 📰 Anthropic 官方 · Econ for Everybody 解读

事件背景：Anthropic 利用其隐私保护数据分析系统，发布了第二份 Claude 经济指数报告（标题为"Learning Curves"）。报告追踪了 Claude 在全经济领域的实际使用演变趋势。

数据亮点：核心发现是，用户使用 AI 模型的时间越长，效率提升越显著——即存在显著的"学习曲线"效应。但报告同时警告，这种"越用越好"的模式可能扩大现有的经济不平等差距：早期采用者和资源丰富的组织将获得不成比例的生产力增益。

🔮 产业影响：该报告为"AI 替代论"提供了更细腻的视角——不是简单的替代，而是"AI 熟练度"正成为新的劳动力分层维度。政策制定者需关注 AI 素养培训的公平分配问题。

NVIDIA 发布 ProRL Agent：面向多轮 LLM 智能体的"Rollout-as-a-Service"RL 训练基础设施 RL/Agent

📅 2026-03-27 · 📰 arXiv 论文 · TechWalker 中文报道

事件背景：NVIDIA 研究团队开源发布了 ProRL Agent，这是一个用于多轮 LLM 智能体强化学习训练的可扩展基础设施，已整合进 NVIDIA NeMo Gym 框架。

技术细节：ProRL Agent 的核心创新是提出"Rollout-as-a-Service"理念——将 AI 的"练习/探索（rollout）"过程从"学习/更新"过程中解耦，通过 API 驱动的分布式架构实现大规模并行 rollout。这种解耦设计使得在软件工程等需要多轮交互的复杂 Agent 任务上，训练效率大幅提升。

🔮 产业影响：该工作补齐了 NVIDIA 在"AI Agent 训练基础设施"层面的最后一块拼图。随着 Agent 成为 LLM 落地的核心形态，RL 训练效率的突破将加速 Coding Agent、Research Agent 等产品的成熟。
