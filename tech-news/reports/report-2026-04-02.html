Anthropic Claude Code 源码泄露：51.2万行代码暴露反蒸馏机制与未发布功能

Fortune / Ars Technica / CNBC 2026-04-01

Anthropic 在发布 Claude Code v2.1.88 时，因 npm 包误含 57MB 未混淆的 source map 文件，导致 51.2万行 TypeScript 源代码被完整泄露。这是继"Mythos"泄露事件后，Anthropic 一周内第二次重大安全事故。

泄露代码揭示了三层反蒸馏机制（输出掺假、隐藏推理、协议隔离）以及多个未发布的实验性功能，包括 KAIROS（后台守护进程）、Buddy（电子宠物模式）和"卧底模式"等。韩国开发者在2小时内将其移植为 Python 版 claw-code，已获超5万 GitHub Star。

💡 行业影响：这起事件被称为"被动开源"的典型案例，正加速社区对 Agent 架构的理解。Anthropic 已公开致歉并声明不会影响核心模型权重安全。
