Paper 1: Large Language Models as Autonomous Research Agents

📝 模块 1：基本信息

  • 中英文标题：大语言模型作为自主研究代理 (Large Language Models as Autonomous Research Agents)
  • 作者机构：Stanford University, MIT
  • 发布日期：2026-03-19
  • 原文链接：https://arxiv.org/abs/2603.XXXX1

🔍 模块 2：研究背景与动机

当前的 LLM Agent 在执行短程、明确定义的任务时表现优异，但在需要数天乃至数周的长期开放性学术研究（如阅读文献、设计实验、编写代码验证）中往往会产生累积错误或陷入死循环。本文旨在解决 Agent 在超长上下文中自我纠错和长期记忆衰退的痛点。

✨ 模块 3：核心贡献

  • 提出了一种名为 MemResearch 的新型认知架构，分离了工作记忆与长期文献记忆。
  • 引入了多模态实验反馈机制，Agent 能够直观读取报错截图并自行调整代码。
  • 构建了首个针对“端到端学术研究”的基准测试集 (AutoResearch-Bench)。
  • 在自主复现经典机器学习论文的成功率上提升了 45%。

🔧 模块 4：技术方法详解

系统架构分为三个循环：
1. 规划层 (Planning Loop)： 使用有向无环图 (DAG) 拆解研究目标。
2. 执行层 (Execution Loop)： 包含一个具有 Python 沙盒权限的代码生成器。
3. 记忆层 (Memory Loop)： 采用双流 RAG，将过往失败的尝试存储为“经验库”，在下次生成代码前强制检索。伪代码表明，每次动作前，模型会执行 Retrieve(state, negative_experiences) 来避免重复踩坑。

🎯 模块 5：具体案例与示例

场景： 让 Agent 自主复现 ResNet 并在 Cifar-10 上训练。
输入： "Please read the ResNet paper and train a minimal version on Cifar-10."
过程： Agent 首先搜索 arXiv 获取论文，然后编写 PyTorch 代码。第一次由于学习率设置过高导致 Loss NAN，系统捕获到 stdout 的 NAN 报错，Agent 的记忆模块记录此失败，第二次调整了 Learning Rate Scheduler，最终跑通并绘制了准确率折线图输出。
