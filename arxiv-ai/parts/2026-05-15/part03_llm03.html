LLM-3 · DISCORD: Distribution Corrected Offline Data Distillation for Reasoning

📅 2026-05-13 🏛 George Mason University arXiv:2605.14071
LLM 蒸馏 数学推理 分布纠偏

① 背景与动机

把强模型的推理能力蒸馏到小模型有两条路：(a) 离线蒸馏（teacher 离线生成 trace，学生做 SFT/KL）—— 样本高效但存在训练-推理分布漂移：训练时学生总是看 teacher 生成的前缀，推理时却必须自回归自己生成的前缀，长 reasoning trace 上误差累积；(b) 在线蒸馏（GKD 等让学生采样自身 rollout）—— 分布匹配但需 ~16× 算力，且训练前期学生质量低拖慢学习。本文要在不引入任何 online sampling 的前提下，纠正离线蒸馏的分布漂移。

② 核心方法详解

① 核心思想：student-support 感知的密度比纠偏

对每条 teacher trace $\{x, y\sim\pi_T(\cdot|x;c)\}$，引入 student/teacher 概率密度比作为重要性权重 $\frac{\bar\pi_\theta(y|x)}{\bar\pi_T(y|x;c)}$（$\bar\pi$ 表示 stop-gradient）。直觉上：teacher 写但学生不会写的 token 权重低，teacher 写学生大概率会自己生成的 token 权重高——把学习火力压在"学生 inference 时容易踩到"的样本上。

② Sigmoid 稳定化 + Token-Level 修正

密度比无界且方差大，作者两步稳定：先做 sigmoid 压缩到 $(0,1)$，再做 token 级而非 sequence 级权重以避免长序列偏置：

$$r_t=\sigma\!\left(\tau\cdot\log\frac{\bar\pi_\theta(y_t\mid x,y_{

$\tau=1$ 时 $r_t$ 退化为 $\frac{\bar\pi_\theta}{\bar\pi_\theta+\bar\pi_T}$，正好等价于"$y_t$ 来自学生分布的后验概率"（uniform prior 下的贝叶斯分类器），这与对比解码（Contrastive Decoding）的训练时对偶。

③ DISCORD 梯度

把 $r_t$ 注入到任何离线 distillation 目标的梯度上，最简单的 SFT 形式为：

这个修正项可作为 SFT、KL、Symmetric-KL 等任意离线 backbone 的插件式 reweighting，不改变其他训练流程。

④ 五阶段 reasoning 提示模板

为了引出 teacher 的高质量推理，作者还设计了一段结构化指令 $c$，把 teacher 的输出强制分成五个阶段：① problem understanding → ② strategy selection → ③ solution → ④ verification → ⑤ final answer。仅保留 teacher 最终答案正确的轨迹用于训练。这样 teacher trace 自身的语义结构更稳定，密度比噪声更小。

③ 深度案例：14B→1.5B 蒸馏中的分布纠偏

场景：用 Qwen2.5-14B-Instruct 作 teacher，把 MATH 推理蒸馏到 Qwen2.5-1.5B-Instruct（学生模型）。
  1. Vanilla SFT：学生在 MATH500 上仅 41.0%，因为 teacher 在某些 token 处用了"Wait, let me reconsider…"等只有大模型自然分布会输出的反思短语，学生从未学会自己说出这类 prefix；
  2. SFT + DISCORD：在 token "Wait" 处 $\bar\pi_\theta(\text{"Wait"})\approx 0.001$，$\bar\pi_T(\text{"Wait"})\approx 0.20$，密度比对数 $\log(0.001/0.20)\approx -5.3$，sigmoid 后 $r_t\approx 0.005$——几乎不学这个 token；而对学生本来就会说的"$x = $"等通用 token，$r_t\approx 0.5$，正常学习。结果 MATH500 从 41.0% → 45.4%（+4.4 pt），GSM8K 从 67.32% → 69.75%；
  3. Symmetric-KL + DISCORD：MATH500 进一步从 44.20% → 50.20%（+6.0 pt），证明纠偏可与不同 backbone 叠加。
关键观察：DISCORD 让学生学会"用学生的方式去说 teacher 的话"，而不是模仿 teacher 自己也说不出来的稀有 prefix；这与对比解码"放大目标模型相对于弱模型的优势"是同一原理的训练时镜像。

⑤ 消融与讨论

跨族实验（Qwen2.5 → Llama-3.2-3B）：在 Qwen2.5-7B 与 14B teacher 下，SFT+DISCORD 在 MATH500 上稳定提升 +2.0~3.0pt（49.40→52.40 / 50.00→52.00），表明纠偏机制不依赖同族 tokenizer。局限：(a) 当学生与 teacher 容量差距过大（如 14B→0.5B）时，密度比噪声主导，sigmoid 截断后大部分 token $r_t\approx 0$，学习信号被抑制；(b) DISCORD 主要解决 token-level 漂移，对句子级"思维链断裂"无能为力，这部分仍需 online rollouts；(c) 对非数学领域（如代码、agent reasoning）的迁移效果未充分验证。

⑥ 结论与启发

DISCORD 用一个轻量的 token-level sigmoid 重权将"在线蒸馏的分布匹配"近似搬到了"离线蒸馏的算力体量"上：用 1/16 的算力，逼近甚至追平 GKD 这类 online baseline。其哲学上的关键是"训练即对比解码"——把推断阶段的"放大学生 vs. 弱模型差异"反过来放到训练阶段做"放大学生 vs. teacher 差异下的反向加权"。对实际工业蒸馏管线的启发是：在已有 offline trace 数据集（最贵的部分已花完）的基础上，几乎不增成本就能把蒸馏效果拉到 online 水平的 80–95%。这是中小团队蒸馏 reasoning 模型的高性价比 baseline。
