📌 今日主线脉络

今日 13 篇精选论文呈现一条清晰的暗线——"RL 让 base 模型变窄"是 2026 年最有争议的核心问题。LLM-4 (Forks-in-the-Road)、RL-2 (NUDGERL)、Agent-1 (Look Before You Leap) 三篇从不同维度直接论证：

  • LLM-4 用 graph navigation 的可控实验证明，决策点存在时 SFT 必然引发 pass@k 衰减，且模型在 fork 处过度自信地走错；
  • RL-2 NUDGERL 用 strategy hint 强制覆盖 long-tail reasoning 模式 + self-distillation 蒸回 base，让模型在 8 rollout 下匹配 GRPO 32 rollout；
  • Agent-1 直接量化 task-only RL 让 ECC（探索覆盖率）下降 9.7 pt，并通过 interleaved exploration reward 把 ALFWorld success 从 72.6% 提到 89.7%。

三篇共同得出"RL 不止优化、也在悄悄抹杀 diversity / exploration"的结论。配套疗法的核心模式相似——在训练时显式注入 diversity 信号 + 蒸馏回 critique-free policy（ICRL 的 token-level reweight 是又一个同模式案例）。这构成今日方法论级别的共识。

📌 RL/GRPO 的设计变量更新

  • RL-1 GPRL：用 $k$-维 General Preference Model 替代标量 reward，直接把 reward hacking 从"事后补救"变为"结构性禁止"。per-dimension advantage normalization + drift monitor 是可直接迁移到 LPO 的两个独立设计 patch。
  • RL-3 Hero / SDS：feasibility-gated reward cliff + curriculum-faded structural bonus 是处理硬约束 + 训练-推理 gap 的成熟模板。
  • Agent-2 ICRL：token-level distribution-calibration reweight ratio $w_t = \pi^S(y_t \mid q) / \pi^S(y_t \mid q, c)$ 是处理 mixed prompt prefix 联合训练的通用工具，可推广到任何"训练时有 hint、推理时无 hint"的算法。

对政昀 RLVR/LPO 研究的直接动作项：(1) GPRL 的 multi-axis advantage 与 LPO 框架结合；(2) NUDGERL 的 inter-intra group advantage 在 LPO 多 prompt rewriting 场景的迁移；(3) ICRL 的 token-level reweight 用于处理任何 conditioned-rollout 训练。

📌 长上下文 / KV Cache 工程的"对齐 attention 几何"哲学统一

LLM-1 (OSCAR INT2 KV cache)、LLM-2 (RTPurbo Sparse Attention)、LLM-3 (ASM Attention-State Memory) 三篇看似互不相关，但共享同一哲学：低成本压缩的关键是对齐下游算子真正使用的几何结构。

  • OSCAR 对齐 $C_Q = Q^\top Q$ 与 $C_S = V^\top S^\top S V$；
  • RTPurbo 对齐 RoPE 低频子空间（仅 16 维即可路由）；
  • ASM 把 prefix attention 的代数结构 $(a_p, Z_p)$ 显式分解、可合并、可外化为字典。

2026 年长上下文 LLM 工程的关键认知升级："通用旋转 / 通用压缩"已经走到尽头，下一阶段必须使用算子感知的目标函数。这一哲学对生产推理栈（vLLM/SGLang）的产品力提升路径有直接指导意义。

📌 Agent / Memory 的产品化结论

Agent-3 (EvoMemBench) 给出选型矩阵：context 充裕用 long-context LLM；context 受限（≤32K）优先 procedural memory（ReasoningBank、AWM 类存 strategy 而非 fact）；事实修订（revision）所有现有 memory 系统都失败。Agent-1 (Look Before You Leap) 提供"explore-then-act"推理范式与对应训练目标。Multimodal-1 (SaaS-Bench) 给出 sober 的产业现状：frontier CUA end-to-end SaaS 完成率仅 1.9-3.8%，70% 失败是 entity missing。三者结合是 2026 H2 agent 产品技术选型的最佳指南。

📌 医疗 LLM 的两类核心命题

Medical-1 SEMA-RAG：把 RAG 升级为"Interpret + Explore + Adjudicate"的三 agent 架构，是医学 RAG 工程级最佳实践。Medical-2 Artificial Intolerance：揭示一个监管级风险——所有 9 个 frontier LLM 都被临床记录中的污名化语言污染，且 RLHF 训练让模型"声明无偏见"但"实际有偏见"——这是知行分离的清晰证据。两篇配套读：前者教医学 LLM如何赢，后者教医学 LLM如何不出事。
