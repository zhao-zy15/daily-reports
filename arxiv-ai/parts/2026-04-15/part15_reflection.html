今日亮点（2026-04-15，20篇论文）：

📊 SFT 不完全学习的系统性研究：今日最重要的发现之一来自 ACL 2026 Oral——SFT 收敛后仍有 ~20% 的训练数据学不会。五大根因（知识缺失、知识冲突、数据矛盾、左侧遗忘、优化不足）的诊断框架对工业界的 SFT 实践有直接指导价值。这提醒我们：盲目增加 SFT 数据量或训练轮数是低效的，应该先做根因诊断再做针对性干预。

🎯 RL 训练的效率与安全并重：三篇 RL 论文分别从不同角度推进：Policy Split 通过双模式探索解决了探索-利用困境（AIME +6.6分）；低秩优化轨迹实现了 RLVR 2.3x 加速；而 Qwen/阿里巴巴的校准崩溃研究则敲响了 RLHF 安全性的警钟——逢迎微调系统性地摧毁校准。值得警惕的是，当前绝大多数 RLHF 模型可能都存在未被察觉的校准崩溃。

🤖 Agent 能力的结构性失衡：Google DeepMind+UMN 的 "Amazing Agent Race" 揭示了一个关键洞察：当前 agent 是"强工具使用者但弱导航者"。Navigation Gap 高达 30-42%，这意味着 agent 研究的下一个前沿不是如何调用工具，而是如何规划工具使用策略。OpenAI+MSR 的 Sanity Checks 进一步表明，agent 输出中 ~30% 存在严重错误（数据泄漏、因果混淆、结果幻觉）。

🔬 物理模拟器 + RL = 科学推理突破：DeepSeek+北大的工作开辟了一个全新范式：让 LLM 通过物理模拟器交互来学习物理推理，IPhO 级别题目从 24% 提升至 42%。这可能预示着 "LLM + 领域模拟器" 的通用范式将成为科学推理的关键路径。

💡 趋势观察：

  • 从"更多数据"到"更聪明的数据"：MathAgent（约束图合成）和可验证工具数据合成都指向高质量数据的结构化生产
  • 从"更大模型"到"更聪明的推理"：Qwen 三角色编排（7B→接近GPT-4o 80%）和 SpecMoE（MoE 推理 2.9x 加速）都在模型效率上做文章
  • "校准"成为新的安全维度：TTDD 的自校准和校准崩溃研究从两个方向推进 LLM 可信度
