① 背景与动机

RLVR（Reinforcement Learning with Verifiable Rewards）能可靠提升 pass@1，但 pass@k 往往不增——这意味着 RLVR 不是真正学到新推理能力，而是把基模本有的某条 reasoning mode 锐化了。已有理论（Wu et al., 2025）证明：标准 reverse-KL RLVR 的empirical support 永远是 reference model support 的子集（Theorem 4.2）——即不可能发现 base model 完全没见过的解。常见对策（弱化 KL、用 forward KL、annealed KL）虽能扩 coverage 但会让训练不稳定。本文目标：保留 reverse-KL 的稳定性同时让 empirical support 真正扩展。

② 核心方法详解

① Pseudo-KL：让 anchor 可以非归一化

标准 RLVR 目标 $J(\theta) = \mathbb{E}_{x,y\sim\pi_\theta}[r(x,y) - \beta D_{KL}(\pi_\theta \| \pi_{\text{ref}})]$ 强制 anchor 是归一化分布。作者引入pseudo-KL：

$$\tilde{D}_{KL}(p \| f) = \mathbb{E}_{y \sim p}\left[\log \frac{p(y)}{f(y)}\right]$$

$f$ 可以是任意 positive unnormalized function。这保留 stationary proportional solution 但允许 anchor 用非归一化的"引导函数" $q(x,y)$ 进行 shaping：

$$J'(\theta) = \mathbb{E}\left[r(x,y) - \beta \tilde{D}_{KL}(\pi_\theta \| q \cdot \pi_{\text{ref}})\right]$$

② 三类 SAGE 引导函数

token-level 因子化：$q(x, y) = \prod_t q(y_t | x, y_{<t})$。Pseudo-KL 展开：

$$\tilde{D}_{KL}(\pi_\theta \| q \cdot \pi_{\text{ref}}) = D_{KL}(\pi_\theta \| \pi_{\text{ref}}) - \mathbb{E}_{x,y\sim\pi_\theta}\!\left[\sum_t \log q(y_t | x, y_{<t})\right]$$

三种 $q$ 设计：(1) Random：均匀噪声；(2) Token-level：基于 token 出现频率；(3) Branch-level（推荐）：基于 token entropy

$$q_{\text{Branch}}(y_t | x, y_{<t}) = 1 + \gamma (H_t - \tau)_+ \quad \text{where} \quad H_t = -\sum_v \pi_\theta(v|x, y_{<t}) \log \pi_\theta(v|x, y_{<t})$$

仅在高熵分支点（推理 fork）激活探索，低熵确定 token 不扰动。

③ Empirical Support Expansion 定理

Theorem 4.3：若 $q(x, y^\star) > 0$ 对某个 $y^\star \in \mathcal{C} \setminus \text{supp}_\epsilon(\pi_{\text{ref}})$（即 reference model 不会生成的正确 completion），SAGE 可以让 $\pi_{\text{SAGE}}$ 把 $y^\star$ 纳入 empirical support。这意味着 SAGE 能真正发现 base model 没见过的解，不是单纯锐化。

④ 在线计算无开销

所有 intrinsic signals（token entropy）可以从 rollout forward pass 中直接拿到，不需要额外计算。SAGE 与 GRPO 算力开销几乎一致，是 plug-and-play 升级。

③ 深度案例

Qwen2.5-Math-7B 在 AIME 上：
  1. Base：pass@1 = 0.033, pass@256 = 0.393。说明 base model 有 39.3% 题目能在 256 次采样中至少找到一次正解。
  2. GRPO 标准训练 1000 步：pass@1 升到 0.053（+0.020 pt），但 pass@256 几乎不变（0.407，+0.014 pt）。这是典型 mode sharpening：把已有 mode 收敛得更紧，但没扩 coverage。
  3. SAGE-Branch：把 q(y_t) 设为 entropy-gated multiplier。当模型在某个 reasoning step 出现高熵（如选择哪个公式应用）时，给予更多探索奖励。训练 1000 步后：pass@1 = 0.063（vs GRPO 0.053），pass@256 = 0.427（vs GRPO 0.407） — 双向都提升。
  4. Branch trigger：典型触发点是"设 x=?"或"应用引理?" 这种 token，模型的 next-token entropy 显著高于普通 token（如 H=2.5 nats vs 普通 H=0.3）。

⑤ 消融与讨论

三个 q 设计的对比：Random（无差别噪声）效果有限，Token-level（基于词频）反而比 GRPO 差，Branch-level（基于 entropy）才显著生效——证明 探索应该集中在推理分支点，而非均匀扰动。γ, τ 敏感性：$\gamma$ 控制扰动幅度、$\tau$ 控制 entropy 触发阈值，论文在 Appendix D 给出 grid search，最优 $\gamma=0.5, \tau=2.0$。局限：仅在 7B 模型 + 数学推理上测试，对长 chain reasoning（如代码、定理证明）需要重新调参；与 Forward-KL 的本质区别需要更深的理论比较——两者都是 KL 调整，但 SAGE 在 token-level 更细。

⑥ 结论与启发

SAGE 提供了 RLVR 探索-利用 tradeoff 的新理论框架：把 KL anchor 从"归一化分布"扩展到"未归一化引导函数"，理论上证明 empirical support 可以严格扩张。这与 LambdaPO（RL-1, pairwise advantage）、DMPO（RL-4, forward KL distribution matching）一同构成了 2026 年 5 月 RLVR 三大方向：分别从 advantage estimator、optimization objective、exploration shaping 三个角度治理 GRPO 的 mode collapse 问题。SAGE 的"在高熵点扰动"实践非常工程友好——它不需要额外网络、不需要 critic、不改变 GRPO loop，仅在 token-level 加一个 entropy-gated multiplier 即可。
