方法范式三股潜流交汇。今天的 14 篇论文呈现出三个相互呼应的方法论趋势：(1) "内部信号驱动"——FoCore 用 JS 散度识别 HD token、T²PO 用 token 熵触发非犹豫事件、SAPO 用熵分段、Component-Aware 用 PPL 消融预测可行性，都把模型内部不确定性作为动态控制信号而非外部奖励；(2) "频谱式优化粒度"——SAPO 段级 RL、Gate-DPO 序列级门控、CoRD 步骤级蒸馏、orchestration trace 事件图，全部呼应"优化粒度应匹配任务结构"的命题；(3) "frozen backbone + inference-time 改造"——Visual Latents Unsilencing、Component-Aware、FoCore 都不动主模型参数，靠推理时的对比、奖励、门控达到训练级提升。

评测范式正经历范式革命。三篇 paper（CLEAR、Reasoning Trap、RMGAP）从不同角度揭示传统 accuracy-only benchmark 的盲区：CLEAR 揭露医学 LLM 缺乏"知道何时不知道"的能力；Reasoning Trap 用信息论证明 multi-agent debate 在保持准确率的同时系统性侵蚀证据接地；RMGAP 显示奖励模型对偏好多元性几乎完全无知。这三个工作共同呼应一个新的研究纲领："reliability beyond accuracy"——模型在认知谦逊、推理忠实度、偏好泛化三个维度上的真实表现，与 leaderboard 数字常常背道而驰。这意味着 2026–2027 学术界与产业界要联手重塑评估标准。

多 Agent 系统进入理论奠基期。2605.02801 综述把 LLM-MAS RL 重写为 dynamic Dec-POMDP + orchestration trace，2605.01704 给出 closed-system 推理的 DPI 上界，2605.01566 给出 multi-agent 测试时算力 Pareto 前沿。三篇互补：综述+理论+实证，标志着多 Agent 系统从"经验工程"开始迈向"形式化科学"。Kimi K2.6 公开 Agent Swarm（≥64 sub-agents）的工业证据使这种理论化变得迫切。下一阶段最值得关注的是：能否把 EGSR、orchestrator-as-policy RL、Pareto-aware 路由这三类工具合并到同一框架中？

RL 后训练的"理论-实践"裂缝在缩小。2605.02469（Reference-Sampled Boltzmann Projection）首次给出 KL-RLVR 与 weighted SFT 严格等价的充要条件，并解释了为何 multi-epoch SFT 永远修复不了 coverage 缺陷。这与 OpenAI/Anthropic 实践经验"必须 sampler refresh"完全一致，但首次有了形式化解释。结合今天的 Gate-DPO（梯度几何）、SAPO（段级粒度）、T²PO（探索控制），可以勾勒出 2026 年下半年的 RL 后训练新范式：从损失函数工程转向梯度动力学 + 采样几何 + 优化粒度的三位一体设计。

潜在主题：把"什么是好"重新变成可量化对象。FoCore 的 HD token、Reasoning Trap 的 SFS（Semantic Faithfulness Score）、CLEAR 的 abstention rate、RMGAP 的 ranking consistency、SAPO 的 segment-level advantage —— 五者都试图把传统上"质性"的概念（核心 token、推理忠实、认知谦逊、偏好稳健、推理步骤）转化为可计算指标。这是 NLP 走向科学化的重要标志：当一个领域的关键概念都可以被精确测量后，工程改进才能有方向。
