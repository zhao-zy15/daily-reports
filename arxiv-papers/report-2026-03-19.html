AgentFactory: A Self-Evolving Framework Through Executable Subagent Accumulation and Reuse

Agent Self-Evolution
👤 Zhang Zhang, Shuqi Lu, Hongjin Qian, Di He, Zheng Liu 📍 Peking University
摘要：构建基于 LLM 的智能体自我进化新范式——将成功经验保存为可执行的子代理代码，而非文本提示或反思。子代理基于执行反馈持续改进，随着任务积累变得更加健壮高效。

🎯 核心贡献

  • 提出可执行子代理概念：成功经验转化为 Python 代码而非文本
  • 实现持续能力积累：子代理库随时间增长和改进
  • 代码可移植性强：纯 Python + 标准化文档，适用于任何环境

🔧 技术方案

  • 存储形式：子代理保存为纯 Python 代码，附带标准化文档
  • 进化机制：基于执行反馈进行代码迭代优化
  • 复用架构：子代理库积累后可快速解决类似任务

💡 研究启示

  • 从文本记忆到代码记忆的转变，提升了 Agent 的可复现性
  • 类似人类技能习得——将经验固化为可重复执行的程序
  • 未来可探索跨领域子代理迁移学习
